# Welcome to Altarya

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

This is an [Expo](https://expo.dev) project created with [`create-expo-app@latest`](https://docs.expo.dev/router/installation/).

I ran `npm run reset-project` to get a fresher start.
